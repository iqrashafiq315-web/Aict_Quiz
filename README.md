# AI Quiz Generator (No API Key)

A simple AI-style quiz generator built using Python and Streamlit without any external API.

---

## Features
- Select topic (Physics, Computer Science, etc.)
- Choose difficulty level
- Generate random MCQs
- Unlimited question generation (with repetition allowed)
- Timer-based quiz system (60 seconds)
- Instant scoring system

---

## Technology Used
- Python
- Streamlit
- Random module
- Time module

---

## Run Locally
streamlit run app.py
